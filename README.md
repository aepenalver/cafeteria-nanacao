# Cafetería Nanacao

Set de pruebas unitarias para aplicación Cafetería Nanacao, como evaluación de prueba de módulo de testing y demostración de conocimiento de Testing Unitarios con Jest y Supertest.

## Tecnologías Empleadas

- Node JS
- JavaScript
- Express JS

## Requisitos

- Tener instalado [Node.js](https://nodejs.org/) (versión 14 o superior).
- Una terminal o consola de comandos.
- Opcional: Tener instalado algún "Cliente API"

## Instalación

1. Clonar repositorio:

```bash
git clone https://github.com/aepenalver/cafeteria-nanacao.git
```

2. Acceder al directorio:

```bash
cd cafeteria-nanacao.git
```

3. Instalar las dependencias:

```bash
npm install
```

4. Uso (modo de pruebas):

```bash
npm run test
```

## Estructura de la aplicación

```bash
.
├── Cafeteria Nanacao.postman_collection.json   # Archivo para importar en Postman con todas las rutas y pruebas listas.
├── cafes.json                                  # Registros
├── index.js                                    # PUNTO DE ENTRADA: Entrada de la app.
├── package.json                                # Configuración del proyecto, scripts y dependencias.
└── tests                                       # CAPA DE PRUEBAS: Tests automatizados para asegurar que el código no se rompa.
```
